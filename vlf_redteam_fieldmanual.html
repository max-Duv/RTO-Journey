PASSIVE MONITORING SETUP PHASE 1

Your first 72 hours are purely passive. No transmissions, no active probing. Build a complete picture of baseline behavior.

DEPLOY RECEIVING ANTENNA

VLF requires large antennas. A simple long-wire (30–100m) or magnetic loop antenna (1–3m diameter) will capture signals effectively. Orient for maximum signal pickup from known transmitter directions.

CONFIGURE SDR + UPCONVERTER

Most SDRs bottom out at 500 kHz. You need a VLF upconverter (e.g., Ham-It-Up, SpyVerter) to shift the VLF band up into receivable range. Set center frequency to 125 kHz (with 125 kHz upconverter offset).

ESTABLISH NOISE FLOOR BASELINE

Run continuous IQ recording for minimum 72 hours. Log signal strength, timestamps, and any burst patterns. Power grid interference at 50/60 Hz harmonics will be heavy — notch filter aggressively.

CORRELATE AGAINST OPERATIONAL PATTERNS

Compare transmission timing against shift changes, facility schedules, environmental events. Predictable transmission windows reveal operational rhythms and confirm active nodes.

// PYTHON — AUTOMATED SIGNAL FINGERPRINT LOGGER
import numpy as np from scipy import signal import datetime, json def fingerprint_burst(iq_samples, sample_rate=250000): """Extract RF fingerprint from captured IQ burst""" # Instantaneous frequency via Hilbert transform analytic = signal.hilbert(iq_samples) inst_phase = np.unwrap(np.angle(analytic)) inst_freq = np.diff(inst_phase) / (2.0 * np.pi) * sample_rate # Power envelope for timing characterization envelope = np.abs(analytic) # Rise time: 10% to 90% of peak power peak = np.max(envelope) rise_start = np.argmax(envelope > peak * 0.1) rise_end = np.argmax(envelope > peak * 0.9) rise_time_us = (rise_end - rise_start) / sample_rate * 1e6 # Center frequency estimation freqs = np.fft.rfftfreq(len(iq_samples), 1/sample_rate) fft_mag = np.abs(np.fft.rfft(iq_samples)) center_freq = freqs[np.argmax(fft_mag)] fp = { "timestamp": datetime.datetime.utcnow().isoformat(), "center_hz": round(center_freq, 2), "rise_us": round(rise_time_us, 3), "mean_freq_hz": round(np.mean(inst_freq), 4), "freq_std": round(np.std(inst_freq), 6), "burst_dur_ms": round(len(iq_samples)/sample_rate*1000, 2) } return fp # Save fingerprint database with open('vlf_fingerprints.jsonl', 'a') as f: f.write(json.dumps(fingerprint_burst(captured_iq)) + '\n')
TDOA GEOLOCATION ACTIVE-ADJACENT
Requires minimum 3 synchronized receiver positions. Use GPS-disciplined receivers or white-rabbit timing for sub-microsecond sync. Position accuracy degrades significantly without precise timing.

Time Difference of Arrival (TDOA) allows triangulation of VLF transmitters using multiple passive receivers — no emissions required from your side.

// TDOA POSITION ESTIMATION — HYPERBOLIC INTERSECTION
from scipy.optimize import minimize import numpy as np # Receiver positions [lat, lon] — GPS surveyed receivers = np.array([ [38.8951, -77.0364], # RX-A [38.9072, -77.0401], # RX-B [38.8820, -77.0198], # RX-C ]) # Time delays in microseconds (from cross-correlation) tdoa = np.array([0.0, 12.4, -8.7]) # relative to RX-A c = 299792458 # m/s def tdoa_cost(pos): """Cost function: minimize sum of squared TDOA residuals""" dists = np.sqrt(np.sum((receivers - pos)**2, axis=1)) * 111320 predicted_tdoa = (dists - dists[0]) / (c * 1e-6) return np.sum((predicted_tdoa - tdoa)**2) result = minimize(tdoa_cost, x0=np.mean(receivers, axis=0), method='Nelder-Mead') print(f"Estimated TX position: {result.x}")

// PYTHON — AUTOMATIC MODULATION CLASSIFIER
import numpy as np from scipy.stats import kurtosis def classify_modulation(iq, fs): """Heuristic modulation classifier for VLF signals""" analytic = iq / (np.abs(iq) + 1e-10) # normalize phase = np.angle(analytic) amplitude = np.abs(iq) inst_freq = np.diff(np.unwrap(phase)) * fs / (2*np.pi) amp_variance = np.var(amplitude) phase_jumps = np.sum(np.abs(np.diff(phase)) > np.pi/2) freq_clusters = len(np.unique(np.round(inst_freq, -1))) if amp_variance > 0.3 and phase_jumps < 10: return "OOK" elif freq_clusters <= 3 and amp_variance < 0.05: return "FSK" elif phase_jumps > 50 and amp_variance < 0.05: return "BPSK/MSK" else: return "UNKNOWN — manual analysis required"

VLF ANTENNA DESIGNS BUILD

Antenna performance is everything at VLF. The wavelengths are enormous (10–100 km), so every antenna is electrically short. Here are your practical options:

MAGNETIC LOOP (RECOMMENDED FOR RECON)

A 1–3 meter diameter shielded loop provides excellent noise rejection and reasonable sensitivity. Wind 5–20 turns of wire on a PVC or wood frame. Connect to a low-noise preamp before the SDR.

// LOOP ANTENNA SPECIFICATIONS
Diameter: 2.0 m Turns: 10 Wire gauge: 18 AWG Inductance: ~180 μH (measured with LCR meter) Resonant cap: C = 1 / ((2π·f)² · L) @ 15 kHz: C = 1/((94247)² · 180e-6) ≈ 627 pF Preamp: OPA627 or LT1028 based, gain +30 dB Input impedance: 10 MΩ balanced Noise figure: <2 nV/√Hz

E-FIELD VERTICAL (LONG WIRE)

Deploy 30–100m of wire horizontally or as an inverted-L. Higher noise pickup than loop but captures signals from all directions. Requires good ground counterpoise.

#!/usr/bin/env python3 """VLF Mesh Network Decoder — Red Team Field Tool""" from gnuradio import gr, blocks, filter, analog, digital from gnuradio.filter import firdes import numpy as np class VLFMeshDecoder(gr.top_block): def __init__(self, center_freq=125000, # Hz (with upconverter) samp_rate=250000, # samples/sec target_freq=15000, # target VLF channel symbol_rate=100, # baud — measure first! output_file='decoded.bin'): gr.top_block.__init__(self, "VLF Mesh Decoder") ## ── SOURCE: RTL-SDR via osmosdr ────────────────── import osmosdr self.src = osmosdr.source(args="numchan=1 rtl=0") self.src.set_sample_rate(samp_rate) self.src.set_center_freq(center_freq) self.src.set_gain(40) self.src.set_if_gain(32) ## ── TRANSLATE: tune to target VLF channel ──────── offset = target_freq - center_freq # freq offset to channel self.xlate = filter.freq_xlating_fir_filter_ccf( decimation=5, taps=firdes.low_pass(1, samp_rate, 2000, 500), center_freq=offset, sampling_freq=samp_rate ) # New rate after decimation: 250000/5 = 50000 Hz ## ── NOTCH FILTER: kill 50/60 Hz power line noise ─ self.notch_50 = filter.iir_filter_ccz( # 50 Hz and harmonics notch coefficients fwd_taps=self._notch_coeffs(50, 50000), rev_taps=[] ) ## ── AGC: normalize signal amplitude ────────────── self.agc = analog.agc_cc( rate=1e-3, reference=1.0, gain=1.0 ) ## ── DEMODULATE: FSK/BPSK — select based on analysis sps = int(50000 / symbol_rate) # samples per symbol self.clock_rec = digital.clock_recovery_mm_cc( omega=float(sps), gain_omega=0.001, mu=0.5, gain_mu=0.05, omega_relative_limit=0.005 ) ## ── DECISION: hard symbol decisions ────────────── self.slicer = digital.binary_slicer_fb() ## ── SINK: save decoded bits ────────────────────── self.file_sink = blocks.file_sink( gr.sizeof_char, output_file ) ## ── CONNECT CHAIN ──────────────────────────────── self.connect( self.src, self.xlate, self.agc, self.clock_rec, self.slicer, self.file_sink ) if __name__ == '__main__': decoder = VLFMeshDecoder(target_freq=15200, symbol_rate=75) decoder.run()

// CUSTOM SCAPY LAYER FOR VLF MESH PROTOCOL
from scapy.packet import Packet from scapy.fields import * import struct, binascii # Define protocol structure from reverse engineering class VLFMeshFrame(Packet): name = "VLFMeshFrame" fields_desc = [ XByteField("sync1", 0xAA), # sync word byte 1 XByteField("sync2", 0x55), # sync word byte 2 XByteField("src_addr", 0x00), # source node ID XByteField("dst_addr", 0xFF), # dest (0xFF = broadcast) ShortField("seq_num", 0), # sequence number ByteField("cmd", 0x01), # command byte FieldLenField("payload_len", None, length_of="payload"), StrLenField("payload", b"", length_from=lambda p: p.payload_len), XShortField("crc16", 0x0000), # CRC-16/CCITT ] def compute_crc(self): """Compute CRC-16/CCITT over frame (excl. CRC field)""" raw = bytes(self)[:-2] # everything except CRC crc = 0xFFFF for byte in raw: crc ^= byte << 8 for _ in range(8): crc = (crc << 1) ^ 0x1021 if crc & 0x8000 else crc << 1 return crc & 0xFFFF # Craft spoofed packet as node 0x03 commanding node 0x07 pkt = VLFMeshFrame( src_addr=0x03, # impersonate node 3 dst_addr=0x07, # target node 7 seq_num=0x00A2, # next expected seq (from capture) cmd=0x05, # e.g., 0x05 = STATUS_REQUEST payload=b'\x00' ) pkt.crc16 = pkt.compute_crc() raw_frame = bytes(pkt) print(f"Crafted frame ({len(raw_frame)} bytes): {raw_frame.hex()}")

MESH PARTITIONING ATTACK ADVANCED

Rather than brute-force jamming, a surgical partitioning attack severs the mesh between two segments by targeting only the relay node(s) connecting them. This is more effective, stealthier, and reveals architectural weaknesses.

// MESH TOPOLOGY INFERENCE FROM PASSIVE OBSERVATION
import networkx as nx import json from collections import defaultdict # Build mesh topology graph from captured packet routing headers G = nx.DiGraph() # Load parsed packet log (from URH/GNU Radio decoder output) with open('packet_log.jsonl') as f: packets = [json.loads(l) for l in f] for pkt in packets: src = pkt['src_addr'] dst = pkt['dst_addr'] if dst != 0xFF: # ignore broadcasts G.add_edge(src, dst, weight=G.get_edge_data(src,dst,{}).get('weight',0)+1) # Find critical relay nodes (high betweenness = partition target) betweenness = nx.betweenness_centrality(G, weight='weight') sorted_nodes = sorted(betweenness.items(), key=lambda x: -x[1]) print("PARTITION TARGETS (by betweenness centrality):") for node, score in sorted_nodes[:5]: print(f" Node 0x{node:02X}: centrality={score:.4f}") # Simulate partition: remove top node and check connectivity top_target = sorted_nodes[0][0] G_severed = G.copy() G_severed.remove_node(top_target) components = list(nx.weakly_connected_components(G_severed)) print(f"\nRemoving node 0x{top_target:02X} creates {len(components)} isolated segments")
TACTICAL NOTE: In a real engagement, once you identify the critical relay node via topology analysis, you have three options: (1) physical disruption of the node hardware, (2) targeted RF denial against that node's frequency/slot only, or (3) protocol-level attack to crash the node's firmware via malformed packets.

DETECTION EVASION — ACTIVE PHASE OPSEC

MIMIC NATURAL INTERFERENCE

VLF is plagued by natural noise: lightning (sferics), power line harmonics, solar activity. Structure your test transmissions to blend with these. Short bursts during thunderstorm activity are nearly impossible to distinguish from natural noise at the receiver side.

POWER MANAGEMENT

Use the minimum power required to reach your target. VLF propagates extremely well — you almost certainly need far less power than you think. Over-powering is the #1 way to get detected. Measure path loss first, then set TX power 3 dB above minimum decode threshold.

TIMING DISCIPLINE

Never transmit in a predictable pattern. Randomize inter-transmission intervals. Avoid daily patterns that could be identified through traffic analysis over multiple days.

ANTENNA DIRECTIONALITY

A directional antenna (end-fire array, beverage antenna) concentrates energy toward the target and away from direction-finding receivers. Reduces your observable RF footprint significantly.

// POWER LEVEL CALCULATOR — MINIMUM VIABLE TX POWER
import math def min_tx_power_dBm( distance_km, freq_kHz, rx_sensitivity_dBm=-120, # typical VLF receiver link_margin_dB=10, # safety margin tx_ant_gain_dBi=0, rx_ant_gain_dBi=0 ): """VLF ground wave path loss model (Norton, 1937)""" wavelength_m = 3e8 / (freq_kHz * 1000) d_m = distance_km * 1000 # Free-space path loss component fspl_dB = 20*math.log10(d_m) + 20*math.log10(freq_kHz*1000) - 147.55 # Ground wave attenuation (simplified, flat earth, avg ground) # Real implementation uses Wait/Norton tables by conductivity gw_atten_dB = 0.0158 * distance_km * math.sqrt(freq_kHz) total_loss = fspl_dB + gw_atten_dB min_tx = rx_sensitivity_dBm + total_loss + link_margin_dB - tx_ant_gain_dBi - rx_ant_gain_dBi return { 'path_loss_dB': round(total_loss, 1), 'min_tx_dBm': round(min_tx, 1), 'min_tx_mW': round(10**(min_tx/10), 3) } # Example: 2 km range at 15 kHz result = min_tx_power_dBm(distance_km=2, freq_kHz=15) print(f"Path loss: {result['path_loss_dB']} dB") print(f"Min TX power: {result['min_tx_dBm']} dBm ({result['min_tx_mW']} mW)") # Result: often <1 mW is sufficient for short-range tests
